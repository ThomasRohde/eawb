# Keyboard Shortcuts

EA Workbench supports keyboard shortcuts for efficient navigation and editing.

## Tree Panel (Rename Mode)

When renaming a capability inline:

| Shortcut   | Action         |
| ---------- | -------------- |
| **Enter**  | Confirm rename |
| **Escape** | Cancel rename  |

## AI Chat

| Shortcut        | Action              |
| --------------- | ------------------- |
| **Enter**       | Send message        |
| **Shift+Enter** | New line in message |

## General

Standard browser shortcuts apply throughout the workbench:

| Shortcut   | Action           |
| ---------- | ---------------- |
| **Ctrl+Z** | Undo (in inputs) |
| **Ctrl+Y** | Redo (in inputs) |
| **F12**    | Open DevTools    |
| **Ctrl+R** | Reload page      |

## Tips

- Most shortcuts are context-sensitive — they apply to the currently focused input or panel
- The tree panel uses a three-dot menu for actions (Add child, Rename, Delete) rather than keyboard shortcuts
- Drag-and-drop is the primary way to reorganize capabilities in the tree
