# Themes

EA Workbench supports **Light**, **Dark**, and **System** themes to match your preference and reduce eye strain.

## Switching Themes

1. Look at the bottom of the **sidebar**
2. Click the theme button (shows the current theme name)
3. Select from:
   - **Light** — Bright background, dark text
   - **Dark** — Dark background, light text (Abyss theme)
   - **System** — Automatically matches your operating system's light/dark setting

## Theme Behavior

### System Theme

When set to **System**, the workbench listens for OS-level theme changes. If you switch your OS from light to dark mode, the workbench follows immediately — no restart needed.

### Panel Consistency

All panels respect the current theme:

- Tree panel, Inspector, Export — all use theme-appropriate colors
- AI Chat — messages render with theme-matched backgrounds
- Editor — text and preview areas adapt to the theme
- Popout windows — inherit the theme from the main workbench

### Persistence

Your theme choice is saved in browser local storage and persists across sessions.

## Accessibility

The dark theme (Abyss) uses high-contrast colors designed for extended use in low-light environments. Both themes maintain WCAG-compliant contrast ratios for text readability.
